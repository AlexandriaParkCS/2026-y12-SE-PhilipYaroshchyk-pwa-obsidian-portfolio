***New User Sign Up Experience

Users can access application's main page only. New users are required to sign up by providing their name, email and password. User name and email are limited to 50 characters. Password requires an upper case, a lower case, a digital and a special character and must be between 8 and 50 characters. Password is masked to prioritise the security. 

Upon successful registration, user is redirected to their dashboard. At this point the user is presented with a page which indicates that the user has no data yet. Links are provided to encourage user to start using the app by creating entries for goals, expenses and goals.

***Existing User Login

Users that have existing accounts can login by providing their user name and password. If user exists and password is correct, user is logged in and is redirected to their dashboard.

![[login1.drawio.png]]
Application shows the error message if the user is unable to login.

***Add New Goal

User can add a new goal by clicking the "Add Goal" menu button at the top of the page. User is required to add the amount, name, start date and the end date of the goal. Once the form is submitted, the user goals section is updated to reflect the changes. The user can further interact with the dashboard.

![[goal.drawio.png]]

***Add New Expense or Income Entry 

The user can add a new expense or income entry by clicking the "Add Transaction" menu button at the top of the page. User is required to provide the amount, select the type of the transaction from the following list: 

- Income - all types
- Groceries
- Eating out
- Memberships
- Transportation
- Utilities
- Rent
- Mortgage payments
- Insurance
- Personal expenses
- Shopping spree / consumerism
- Investments
- Travelling / Holidays
- Gas
- Debt payments
- Hobbies
- other necessities


![[transaction.drawio.png]]
***Summary Page

The user can navigate to the summary page by clicking on the corresponding summary link in the User Goals section of the dashboard. 

Summary page contains four sections: 
- The goal summary that includes the target amount, current total and the deadline. 
- The transaction line chart that shows expenses and income entries over the goal lifetime
- The pie chart that shows the total of all expenses for each expense type 
- The insight section has summarises how much more savings is required to achieve the goal
